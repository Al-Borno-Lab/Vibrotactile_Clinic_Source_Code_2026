# README

## Introduction - What is this toolkit for? 
- This toolkit is developed to parse Medtronic Percept PC DBS JSON exports. 
- The format of the JSON file could be hard to manipulate.
- Due to the way measured brain LFP are packeted, it could make parsing them into time-series data more tricky. Thus this toolkit aims to simplify that process. 

## General Notes
- My goal is develop this functions to have a standard interface of accepting dictionaries.
